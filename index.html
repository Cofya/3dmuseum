Халцедон - полупрозрачная разновидность кварца различных цветовых оттенков. Шпат - минерал, обладающий способностью раскалываться при ударе по нескольким направлениям спайности.
  • Из группы силикатов: полевой шпат.
  • Из группы карбонатов: известковый шпат (или кальцит), исландский шпат (разновидность кальцита).
  • Флюорит - плавиковый шпат
Сингония - в кристаллографии группа видов симметрии, в которую входят кристаллы, имеющие сходные элементы симметрии.
Пример:
  • Триклинная сингония
  • Моноклинная сингония
